# One Piece Sniper Dashboard

Standalone static dashboard for the One Piece hybrid sniper cron.

## What it does
- Parses the markdown output from `reports/one-piece-hybrid/scan-YYYY-MM-DD.md`
- Displays tracked cards, all-in shipped prices, buy-under thresholds, projected spread, recommendation, confidence, and listing links
- Saves local notes in browser storage

## Local use
Open `index.html` directly in a browser and load a scan report.

## Vercel deploy
This folder is static and Vercel-ready.
- Root directory: `one-piece-sniper`
- Framework preset: `Other`
- No build command required
- Output directory: `.`
